Informações sobre o sistema

— Qual é o projeto? —

O projeto consiste no desenvolvimento de um software de votação online que permitiria votar a partir de um dispositivo conectado à Internet, pode ser uma rede interna e controlada pela instituição que organiza as eleições. Dessa forma, o eleitor não deverá deslocar-se ao local de votação e poderá votar pela rede.

— Qual é a ideia? —

Formulamos entre a equipe de trabalho diversos programas inovadores e de grande interesse para a comunidade, entre eles o Sistema de Voto formulado por uma de suas idealizadoras, Gabriela Barrera, que propôs o sistema de votação eletrônica, já que atualmente o país não possui. e seria de grande interesse para a população poder votar de forma eletrônica que facilitasse o usuário e sua forma de voto.

— Função —

O programa verifica a entrada dos dados eleitorais, solicitando dados pessoais e verificando se eles estão aptos a votar, permite a seleção de uma das opções apresentadas, e verifica se é a escolha correta, através deste voto os gráficos serão atualizados, no o número de votos apresentados por cada seleção e os votos de cada departamento.

Perguntas frequentes

O que devo fazer se a instalação não iniciar?

É provável que tenhamos desativado o programa porque estamos fazendo manutenção ou pode ser um problema de conexão.

O que devo fazer se o sistema não iniciar após a instalação?

Verifique se o seu sistema operacional é compatível com os requisitos que o programa pede. Se não for o sistema operacional do seu dispositivo, pode ser um erro nos frameworks instalados, atualize-os para a versão necessária. Se não houver nenhum, entre em contato conosco!

O que devo fazer se não conseguir lembrar a senha?

Não se preocupe, o sistema pode enviar a você uma nova senha para o e-mail vinculado e se no caso de você perder o e-mail, entre em contato conosco e estabeleceremos para onde enviaremos uma senha de verificação para um novo e-mail.
